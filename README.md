# VCM Data Management System

An open-source web application designed to simplify household registration, caregiver tracking, immunization monitoring, and reporting for community health programs.

## Features

- Household Registration
- Caregiver Management
- Child Registration
- Immunization Tracking
- Dashboard Analytics
- Report Generation
- Excel Export
- PDF Export
- Role-based Authentication

## Tech Stack

Frontend
- React
- Tailwind CSS

Backend
- Node.js
- Express

Database
- PostgreSQL

Authentication
- JWT
